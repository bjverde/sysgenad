SysGenAd para Adianti com FormDin 5


SysGenAd é um software livre. Gerador de sistemas do tipo Scaffold. Criando a estrutura do sistema em poucas etapas:

  • Informe os parâmetros iniciais
  • Informa os dados de conexão no banco de dados
  • Selecione as tabelas, views e storage procedures
  • Sistema Pronto

O que é Scaffold ? A tradução literal do inglês para o português é um tipo de andaime ou plataforma de apoio para construção civil. No desenvolvimento de software é uma técnica antiga que consiste em gerar código baseado em um templete. O código gerado pelo SysGenAd certamente não é o seu sistema final, é apenas um apoio para construção do seu sistema. Diminuindo o tempo total de desenvolvimento. Veja o StackOverFlow para mais explicações https://pt.stackoverflow.com/questions/119731/o-que-%C3%A9-scaffold

O sistema gerado segue o padrão do FormDin usado o Adianti FrameWork 7.5.1.


No GitHub do projeto terá acesso documentação completa, atualizada e issues https://github.com/bjverde/sysgenad

O FormDin 5!

O que é o FormDin ?
é um FrameWork PHP fácil de usar, para criar sistemas web de forma rápida e fácil. Está disponível no portal do software publico brasileiro, no GitHub tem as últimas verões com atualizações e correções de erros.



o Adianti não é um FrameWork PHP ?
Sim o Adianti é um framework PHP MUITO BOM criado pelo Pablo Dall'Oglio.



São dois FrameWork ?
É quase isso !! O FormDin 5 é um meta FrameWork ou um adaptador ou uma extensão, por ser construído sobre o Adinati FrameWork 7.X do Pablo Dall'Oglio. Portando não é um FrameWork completo, ele tem total dependência do Adinati. É um Framework de transição do FormDin 4 para o Adianti FrameWork, facilitando a migração. É uma abstração das chamadas do FormDin 4 no Adianti.

O Adianti é um FrameWork muito bom, com mais recursos e uma comunidade muito maior. Logo o ideal é juntar forçar pegando o que tem de melhor dos dois.
