SysGen para Adianti com FormDin 5


SysGenAd é um software livre, criado usando Adianti FrameWork 7.2. É gerador de sistemas do tipo Scaffold. Criando um sistema em poucas etapas:

  • informe os parametros iniciais
  • Informa os dados do banco
  • Teste a conexão
  • Selecione as tabelas, views e storage procudures
  • Sistema Pronto

    • Scaffold em ingles quer dizer esquelo ou suporte ! O sistema criado pelo SysGenAd é o ponto de partido do sistema, diminuindo o tempo de codifição da versão inicial.

    O sistema gerado usado o FrameWork Adianti usando o FormDin 5 como padrão para as telas.

    O FormDin 5!
    Existe o FrameWork FormDin 4 ! O FormDin 5 é um meta framework ou adpatiador. São as mesma assinaturas e comportamentos do FormDin4 porém o resultado final sempre será uma tela BootStrap do Adianti.
